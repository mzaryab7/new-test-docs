# WireFluid Network Details

This section provides the essential network configuration required for developers to connect applications, wallets, and tooling to the WireFluid blockchain.

**Network Type:** Testnet

**Chain Information**

* **Network Name:** WireFluid Testnet
* **EVM Chain ID:** `92533`
* **Native Token:** WIRE

**RPC & API Endpoints**

* **JSON-RPC Endpoint:** `https://evm.wirefluid.com`

**Block Explorer**

* Explorer URL: `https://wirefluidscan.com`

WireFluid is fully EVM-compatible, allowing developers to deploy and interact with smart contracts using standard Ethereum tooling such as MetaMask, Hardhat, Foundry, and Remix without modification.

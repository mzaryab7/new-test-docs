# WireScan

**WireScan** is the official blockchain explorer for the WireFluid network, designed to inspect, verify, and analyze all on-chain activity in real time. It provides full visibility into blocks, transactions, smart contracts, tokens, and accounts on the WireFluid EVM chain.

WireScan delivers an Ethereum-style explorer experience while leveraging WireFluidâ€™s instant finality and low-fee execution.

### Overview

WireScan enables developers, validators, and users to:

* Monitor network activity
* Inspect EVM transactions and smart contracts
* Track token transfers and balances
* Verify blocks and validator activity
* Debug and audit on-chain behavior

### Key Features

#### Transaction Explorer

* Search by **transaction hash**, **address**, **block number**, or **token**
* View transaction status, gas usage, fees, and execution details
* Instant finality, transactions are final after one block

#### Block Explorer

* Inspect finalized blocks in real time
* View block height, timestamp, proposer, and transaction count
* Track network health and block production

#### Address & Account Tracking

* View EVM account balances in **WIRE**
* Inspect incoming and outgoing transactions
* Monitor contract and externally owned accounts (EOAs)

#### Smart Contract Insights

* Verify and inspect deployed Solidity contracts
* View contract bytecode and transaction interactions
* Debug contract calls and events

#### Token Explorer

* Track ERC-20 compatible tokens
* View token holders, transfers, and supply
* Inspect native and bridged assets

### Network Statistics

WireScan provides high-level network metrics, including:

* Latest finalized block
* Total transactions processed
* Network utilization
* Gas price tracking
* Transaction history and trends

These metrics help developers and validators monitor network performance and usage.

### Developer Use Cases

WireScan is an essential tool for builders on WireFluid:

* Debug smart contract deployments
* Verify transaction execution
* Track contract events and logs
* Inspect gas consumption and fee behavior
* Share transaction or contract links for audits and reviews

### User Use Cases

End users can use WireScan to:

* Verify payment transactions
* Track WIRE transfers
* Confirm staking and reward activity
* Explore on-chain activity transparently
